# UI/UX IMPROVEMENT PROMPT FOR VYAPAAR AI

Analyze the complete existing frontend before making any UI changes. Preserve all existing functionalities and routes.

## SIDEBAR IMPROVEMENTS

Remove the permanently visible left sidebar.

Instead implement a modern AI Operating System style navigation similar to:

* ChatGPT
* Cursor
* Notion
* Perplexity
* Apple Vision OS
* Google Gemini
* Claude AI

### DESKTOP UI

The sidebar should NOT always be visible.

Requirements:

* Completely hide the sidebar by default.
* Add a small menu icon (hamburger icon) in the navbar.
* When the user hovers near the left edge of the screen (5px-10px area), smoothly reveal the sidebar.
* When the mouse leaves the sidebar, automatically collapse it after 500ms.
* Clicking the menu icon should pin/unpin the sidebar.
* Users should also be able to expand/collapse the sidebar manually.

### SIDEBAR ANIMATIONS

Add premium animations:

* Smooth slide animation.
* Glassmorphism effect.
* Blur background.
* Rounded corners.
* Soft shadows.
* Spring animations.
* Hover effects.

Animation requirements:

* 300ms smooth transition.
* Framer Motion animations.
* No lag.
* Responsive transitions.
